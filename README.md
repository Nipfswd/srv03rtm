**NaznaOS Disclaimer**

NaznaOS is an independently developed operating system project created for educational and experimental purposes.

While NaznaOS may appear visually or structurally similar to other operating systems (including but not limited to srv03rtm), it is a **clean-room recreation**. This means:

* No proprietary source code, leaked materials, or unauthorized internal assets were used at any stage of development.
* All components have been written from scratch or built using publicly available, legally obtained resources.
* Any similarities in design, behavior, or structure are the result of independent reimplementation, reverse engineering of publicly observable behavior, or common design patterns—not copying or reuse of protected material.

NaznaOS is **not affiliated with, endorsed by, or derived from any proprietary or leaked source**.

The purpose of this project is to explore system design, compatibility, and historical software architecture in a lawful and ethical manner.
